Google Docs to Adapt

Download a Google Docs file as an HTML and insert it below. It will convert every H2 with the accompanying. Paragraph text below into blocks and respective child components.

IMPORTANT:

  • Make sure the title tags are the same in your google document - each title tag will repersent a block
  • Remember to change the id of the article.js file found in your exported adapt document to 'a-50'

Unsplash Searcher

Type in a keyword of the text area below set the size you want it to be and the number of images you want to get back a text file will accompany the downloaded images with all the attribution and referencing you need.

Narrative component: 1100 x 740 || Everything else: 1100 x 1100

Tracking id Fixer

Update the tracking IDs of all the blocks in numerical order to ensure they are unique

Parent updater

Set the block title to the title of its child component and set all graphic titles to the title of its sibling with the additional 'Graphic for' at the start of the title

Article Assessment Namer

Quickly set all of the assessment ids to the names of a the first child block it finds.
